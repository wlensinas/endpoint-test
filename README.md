# Intro

This is an app in nodejs for testing porpouse.

## Endpoints

* GET '/' to localhost:port -> Hola
* POST '/' to localhost:port -> req.body

## Install

### first option
1. git clone
2. `docker-compose up -d`

### second option
1. git clone
2. `npm install`
3. `npm start` or `npm run start:dev`
